Detailed Description

Capture binary messages.

If a message is longer than one node, additional nodes will be allocated from the pool to create a "jumbo" message.

Calls to record() can be chained together: Blog(NOTE, "foo").record(some_bytes, some_bytes_size).record(more_bytes, more_bytes_size)

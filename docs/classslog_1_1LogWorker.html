◆ start()

void slog::LogWorker::start ( )

Start the worker. If already started, this has no effect. This does not change the signal state to SLOG_ACTIVE. If the state isn't SLOG_ACTIVE, the work loop will run once and the std::thread entrance function will exit, leaving the work thread in an immediately joinable state.

◆ stop()

void slog::LogWorker::stop ( )

Drain any queued messages. Join the work thread. Delete all channels.

Note
Because all work loops are looking at the same atomic stop condition, this will stop all workers.

Detailed Description

A log sink compatible with syslog. Messages are sent over unix sockets or internet sockets.

By default, this logger sends RFC5424-formatted records over UDP. When sending TCP messages, it uses the RFC 6587 "Octet Counting" scheme, where the syslog message is preceded by the byte count to be sent, e.g. "57 <12> 2000-01-01T00:00:00Z mydomain.com my_app Hello World" for RFC3164 formatting.

Note
This can only send messages up to 64kB if the protocol is UDP/IP. Longer messages will be silently truncated. Messages sent to a unix socket or over TCP/IP do not have this limitation.
